# Contributing to Pinterest Search Scraper

Thank you for your interest in contributing! This repository contains documentation for the Pinterest Search Scraper Actor on Apify.

## How to Contribute

Since this is a documentation repository for an Apify Actor, contributions are welcome in the form of:

- Documentation improvements
- Bug reports
- Feature requests for the Actor
- Example use cases

## Reporting Issues

If you find a bug or have a feature request, please:

1. Check existing issues first
2. Create a new issue with:
   - Clear title and description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Environment details

## Documentation Updates

To improve the README.md:

1. Fork the repository
2. Create a branch for your changes
3. Make your improvements
4. Submit a pull request

## Code of Conduct

Be respectful, constructive, and inclusive in all interactions.
